# hackatweet-front
Un projet pour créer un clone de TWITTER (X). Créé et codé par Sébastien ARDAUD et Philip TRAAS. 
La page d'accueil imite le site de Twitter, et a été créée à partir de zéro en utilisant react, css, et redux.

![image](https://github.com/Sebastien-Arbaud/hackatweet-front/assets/135143980/b7f16c46-06d6-4698-b369-3f78c0592ed5)

Le backend a été créé en Node.Js et les données ont été stockées dans MongoDB.

![image](https://github.com/Sebastien-Arbaud/hackatweet-front/assets/135143980/4fccbeb9-9900-4e28-9d09-7a5ec5c3337d)


La conception et la fonctionnalité ont été achevées à environ 90 % pendant la durée du projet.
